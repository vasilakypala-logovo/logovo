СЕЙФ АРТЕФАКТОВ НЕКСА — простой Windows-вариант

Как пользоваться:

1. Откройте файл:
   START_SAFE.hta

2. Нажмите:
   Открыть папку PASSPORTS

3. Положите туда файлы паспортов.

4. Вернитесь в окно сейфа и нажмите:
   Обновить счёт

5. Если нажать "Вынуть из сейфа", файл НЕ уничтожается.
   Он переносится в папку:
   REMOVED_FROM_SAFE

Важно:
- Это работает как Windows HTA-программа.
- Никакой Python не нужен.
- Чёрного окна быть не должно.
- Вся папка NEX_SAFE_HTA переносится целиком на флешку или другой ПК.
