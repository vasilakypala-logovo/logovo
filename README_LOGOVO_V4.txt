ЛОГОВО v4

Что добавлено:
1. Белая планета на первой странице ведёт в Комнату Некса.
2. Комната Некса возвращает в начало ЛОГОВА.
3. В разделе СВЯЗЬ Комнаты Некса есть перенос памяти: сохранить JSON / загрузить JSON.
4. В JSON попадают записи localStorage: подписи галерей, истории работ, сейф артефактов, личные связи.
5. Галереи получили автосохранение при вводе текста.

Как переносить на другой ПК:
— открыть Комнату Некса;
— открыть узел СВЯЗЬ;
— нажать СОХРАНИТЬ ТЕКСТЫ В ФАЙЛ;
— на другом ПК открыть эту же систему, затем ЗАГРУЗИТЬ ТЕКСТЫ ИЗ ФАЙЛА.

Важно: изображения и HTML переносятся всей папкой ЛОГОВО, а JSON переносит изменяемые текстовые записи.
